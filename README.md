# Sistema de Gestión de Imágenes

## Descripción
Este proyecto consiste en el desarrollo de un sistema que permitirá gestionar imágenes de forma sencilla.

## Objetivo
El objetivo del sistema es permitir a los usuarios subir, visualizar, buscar y eliminar imágenes.

## Funcionalidades iniciales
- Subir imágenes
- Visualizar imágenes
- Buscar imágenes
- Eliminar imágenes

## Tecnologías
- Lenguaje: Go
- Control de versiones: Git y GitHub
